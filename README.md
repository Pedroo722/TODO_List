# Como Executar
### Pré-requisitos

- **Typescript**
- **React Native**

### Passos para execução

1. Clone o repositório:
   ```bash
   git clone https://github.com/Pedroo722/TODO_List.git
   cd TODO_List/
   ```
2. Execute o projeto:
   ```bash
   npm run start
   ```
